# Lyme-disease-Elastic-Net-regression-Nowcasting
Using data from regional search trends of Google to fit elastic net regression nowcasting models for regional Lyme disease.

This is the code and data used to develop and fit elastic net regression nowcasting models for Regional Lyme disease using monthly proportional hit data from Google. 
Please use the annotations in the PLOS_Lyme_Nowcasting.R file to guide use

The objective of this reserach was to develop accurate nowcasting models using the widely accessible and free data from Google search database. 

Models fit using this data were accurate and had low levels of error at the regional level and can be found the "Model RDAs" folder.
